# Dotfiles

All of the configs.

## Installation

* Install [rcm][]

* Clone to `~/.dotfiles`

* View what would be installed:

```
$ RCRC="config/rcrc" lsrc
```

* Install it:

```
$ RCRC="config/rcrc" rcup
```

To modify how or what will be installed, see `man 1 rcup`.

[rcm]: https://github.com/mike-burns/rcm
